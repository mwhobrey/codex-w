# @codex/game-systems

Per-RPG plugins. Each subdirectory exports a `GameSystemPlugin`:

- `loner/` — Loner solo RPG (oracle engine)
- `totv/` — Thousand Year Old Vampire (prompt-journal engine)
- `snallygaster/` — Camp Snallygaster (Lasers & Feelings)
- `muscadines/` — Midnight Muscadines (mentor + oracle)
- `ironforge/` — Ironforge (original grim industrial vow-progress)
- `generic/` — System-agnostic fallback sheet

Solo prompt/table text in this package is **codex-w authored** where noted — inspired by published game structures, not reproduced verbatim.

See interface definition in `.cursor/runbook/01_ARCHITECTURE.md`.
