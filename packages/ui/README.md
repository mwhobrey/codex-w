# @codex/ui

shadcn/ui-style components on Codex design tokens.

**Exports:** `Button`, `Input`, `Textarea`, `Label`, `Select`, `Checkbox`, `Card`, `Badge`, `Separator`, `cn`

**Styles:** Import `@codex/ui/styles.css` in the app globals (maps Codex palette → shadcn semantic tokens).

```tsx
import { Button, Card, CardContent, Input } from '@codex/ui';
```
